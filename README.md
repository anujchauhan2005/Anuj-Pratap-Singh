---

## 🎯 What I Learned & Implemented

### Core Concepts Covered:

- **Multi-Head Self Attention** (implemented from scratch)
- **Convolutional Neural Networks (CNN)**
- **Positional Encoding**
- **Encoder-Decoder Architecture**
- **Transformer Blocks** (Self-Attention + Cross-Attention + Feed Forward)
- **Custom Training Loops** with Masked Loss
- **Text Vectorization & Tokenization**

### Key Projects:

1. **Language Translation Transformer**
   - Built a Transformer model from scratch for English to French translation
   - Implemented Multi-Head Attention, Positional Encoding, and Custom Learning Rate Schedule

2. **AI Text Summarizer**
   - Fine-tuned T5 model for abstractive summarization
   - Added multiple summary length modes (Very Short, Short, Long)

3. **UPI Fraud Detection**
   - Machine Learning system to detect fraudulent transactions in real-time
